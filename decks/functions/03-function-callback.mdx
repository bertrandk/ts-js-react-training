# Function Assignment

- Functions can be assigned to variables
- Functions that are assigned to variables can be invoked like you would expect

---
export default FullScreenCode

```javascript
// here is a normal function declaration
function myRegularFunction(a, b) {
  return a + b;
}

// here we are creating a new variable,
// and assigning myRegular
const copiedRegFunction = myRegularFunction;

// assigns the RESULT (return statement)
// of an invoked function
const resultOf3Plus4 = myRegularFunction(3, 4);

// invokes the myRegularFunction
copiedRegFunction(5, 10);
```

## Function Reference vs. Invocation

- Functions are run by invoking them using `()`
- Assigned functions can be referenced without being invoked

---

export default Code

```javascript Function Invocation
function logMyName(name) {
  console.log(name)
}

const returnName = function() {
  return 'Evan';
}

logMyName(returnName()) // This will pass "Evan" to logMyName
logMyName(returnName) // This will pass the function itself
```

---

export default Code

```javascript Function Invocation
function callMyFunction(f) {
  console.log('Calling my function...')
  f()
}

const logName = function() {
  console.log('Evan')
}

callMyFunction(logName) // This will work
callMyFunction(logName()) // This will not
```

## Functions as Arguments / Callbacks

- If we can assign a function to a variable, this means we can assign a function as an argument
- In other words, you can pass functions to other functions
- Parameters that are functions are often referred to as callbacks

---

export default Code

```javascript Function Callbacks
// accept a number, and a callback
function basicCallback(number1, number2, callback) {
  // I am passing the number into the callback, and assigning the result
  const result = callback(number1, number2);
  // and here I return the result
  return result;
}

// a little function that doubles a number and returns the result
const add = (number1, number2) => number1 + number2;
const multiply = (number1, number2) => number1 * number2;
const subtract = (number1, number2) => number1 - number2;

const added = basicCallback(10, 5, add);
const multiplied = basicCallback(10, 5, multiply);
const subtracted = basicCallback(10, 5, subtract);
console.log({added, multiplied, subtracted})
```

```javascript Solution - Function Assignment and Callback 
let toUpperCase = (text) => text.toUpperCase();
let toLowerCase = (text) => text.toLowerCase();

let transform = (transformer, text) => transformer(text);

let changeCase = (toUpper, text) => {
  let transformer = toUpper ? toUpperCase : toLowerCase; 
  return transform(transformer, text)
}


console.log(changeCase(true, 'hElLo WoRld'));
console.log(changeCase(false, 'hElLo WoRld'));
```
