```javascript Array Modification & Mutation
const fruits = ['Peaches', 'Pears'];

fruits.push('Apples'); // Add item to end of array
fruits.unshift('Oranges'); // Add item to start of array
fruits[4] = 'Bananas'; // Add item using an index

moreFruits = fruits.push('Watermelon');
```

```javascript Safely Operating
// Slice - The slice() method returns a shallow copy of a portion of an array into a new array object 
const numbers = [1, 2, 3, 4, 5];
const numbersTwo = numbers;
const numbersThree = numbers.slice(3);

numbers.push(6);

console.log({ numbers, numbersTwo, numbersThree });

const numbersFour = [...numbers, 11];
numbersFour.push(10);
console.log({ numbers, numbersFour })
```

---


## What happened?

- Arrays cannot be copied through assignment
- `push` mutates `numbers`, which is the same as `numbers2`
- `Array.slice` with no indexes creates a copy of the array, so it is not modified
- Spreading the array into a new array creates a copy, so it is not modified
